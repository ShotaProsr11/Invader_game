# Space Invaders Game

シンプルなスペースインベーダーゲームです。Pygameを使用して作成されています。

## 必要条件

- Python 3.x
- Pygame

## インストール方法

1. 必要なパッケージをインストールします：
```bash
pip install -r requirements.txt
```

## 実行方法

以下のコマンドでゲームを起動します：
```bash
python main.py
```

## 操作方法

- 左矢印キー: 左に移動
- 右矢印キー: 右に移動
- スペースキー: 弾を発射
- ESCキー: ゲーム終了
